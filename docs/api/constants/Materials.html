Material（材质）常量

这些常量定义了所有 material（材质）类型的公共属性。 除了 Texture Combine Operations（纹理结合操作），其仅适用于 [page:MeshBasicMaterial.combine MeshBasicMaterial]， [page:MeshLambertMaterial.combine MeshLambertMaterial] 和 [page:MeshPhongMaterial.combine MeshPhongMaterial]。

Side（面）

THREE.FrontSide THREE.BackSide THREE.DoubleSide
设置哪一面会被渲染 —— 前面，后面或双面。 默认值为 [page:Constant FrontSide].

颜色

THREE.NoColors THREE.FaceColors THREE.VertexColors
[page:Constant NoColors] 为默认值，对所有的面使用 material（材质）的颜色。
[page:Constant FaceColors] 根据每个 [page:Face3 Face3][property:Color color] 值对面着色。
[page:Constant VertexColors] 根据每个 [page:Face3 Face3][property:Array vertexColors] 值对面着色。此类型为三个 [page:Color Color] 组成的数组，每一个都对应面的一个 vertex （顶点）。
查看示例 [example:webgl_geometry_colors geometry / colors]。

Blending Mode（混合模式）

THREE.NoBlending THREE.NormalBlending THREE.AdditiveBlending THREE.SubtractiveBlending THREE.MultiplyBlending THREE.CustomBlending
当使用 WebGL 时传入 WebGLRenderer 的 material（材质）的 RGB 和 Alpha值时，这些值控制着 blending equations（混合等式）的源和目标。
[page:Constant NormalBlending] 为默认值。
注意，当使用 [page:CustomBlendingEquation Custom Blending Equations] 时，必须设置 [page:Constant CustomBlending]。
查看示例 [example:webgl_materials_blending materials / blending]。

Depth Mode（深度模式）

THREE.NeverDepth THREE.AlwaysDepth THREE.LessDepth THREE.LessEqualDepth THREE.GreaterEqualDepth THREE.GreaterDepth THREE.NotEqualDepth
material （材质）用来比较当前的像素 Z-depth 缓存值和传入的像素 Z-depth 值的函数。若比较结果为 true 则将会绘制该像素。
[page:Materials NeverDepth] 永不返回 true。
[page:Materials AlwaysDepth] 始终返回 true。
[page:Materials LessDepth] 返回 true，若传入的像素 Z-depth 值小于当前的 Z-depth 缓存值。
[page:Materials LessEqualDepth] 为默认值，若传入的像素 Z-depth 值小于或等于当前的 Z-depth 缓存值时返回 true。
[page:Materials GreaterEqualDepth] 返回 true，若传入的像素 Z-depth 值大于或等于当前的 Z-depth 缓存值。
[page:Materials GreaterDepth] 返回 true，若传入的像素 Z-depth 值大于当前的 Z-depth 缓存值。
[page:Materials NotEqualDepth] 返回 true，若传入的像素 Z-depth 值不等于当前的 Z-depth 缓存值。

Texture Combine Operations（纹理结合操作）

THREE.MultiplyOperation THREE.MixOperation THREE.AddOperation 设置 [page:MeshBasicMaterial.combine MeshBasicMaterial]， [page:MeshLambertMaterial.combine MeshLambertMaterial] 和 [page:MeshPhongMaterial.combine MeshPhongMaterial] 材质时，定义了物体表面的颜色如何与 environment map（环境地图）（若存在）结合。
[page:Constant MultiplyOperation] 为默认值，将 environment map（环境地图）颜色与物体表面颜色相乘。
[page:Constant MixOperation] 使用 reflectivity （反射）混合两种颜色。
[page:Constant AddOperation] 叠加两种颜色。
