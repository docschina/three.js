Draw Mode（绘制模式）常量

[page:Mesh.drawMode] 的不同值控制着 GPU 如何解析传入的 vertices（顶点）组。

注意，仅在 [page:Mesh.geometry] 为 [page:BufferGeometry] 时生效。 当 [page:Mesh.geometry] 为 [page:Geometry] 时无效。

Draw Mode（绘制模式）

THREE.TrianglesDrawMode
默认值，每三个连续 vertices（顶点）(v0, v1, v2), (v2, v3, v5), ... 会被解析为一个独立三角形。
若 vertices（顶点）的数量不是 3 的倍数，则多余的 vertices（顶点）将被忽略。
THREE.TriangleStripDrawMode
一系列三角形以一条线连接起来，例如 (v0, v1, v2), (v2, v1, v3), (v2, v3, v4), ... 后一个三角形与前一个三角形共享两个 vertices（顶点）。
THREE.TriangleFanDrawMode
一系列三角形共享第一个 vertex（顶点）（如同风扇）， 例如 (v0, v1, v2), (v0, v2, v3), (v0, v3, v4), ...

注意： 此模式不支持 [link:https://en.wikipedia.org/wiki/DirectX#DirectX_10 DirectX10]。因 Windows 下 Chrome 和 Firefox 使用 [link:https://en.wikipedia.org/wiki/ANGLE_(software) ANGLE] 渲染 WebGL。 在内部此模式可能会被转换为支持的模式， 在某些浏览器下可能导致低性能。

用法

var geometry = new THREE.Geometry(); geometry.vertices.push( new THREE.Vector3( -10, 10, 0 ), new THREE.Vector3( -10, -10, 0 ), new THREE.Vector3( 10, -10, 0 ), ... ); geometry.faces.push( new THREE.Face3( 0, 1, 2 ), ... ); var material = new THREE.MeshBasicMaterial( { color: 0xffff00 } ); var mesh = new THREE.Mesh( geometry, material ); mesh.drawMode = THREE.TrianglesDrawMode; //default scene.add( mesh );
