自定义 Blending Equation（混合等式）常量

示例

[example:webgl_materials_blending_custom materials / blending / custom ]

用法

适用于所有 material（材质）类型。首先设置 material（材质）的混合模式为 THREE.CustomBlending，然后设置对应的 Blending Equation（混合等式）, Source Factor（源参数）和 Destination Factor（目标参数）。 var material = new THREE.MeshBasicMaterial( {color: 0x00ff00} ); material.blending = THREE.CustomBlending; material.blendEquation = THREE.AddEquation; //默认值 material.blendSrc = THREE.SrcAlphaFactor; //默认值 material.blendDst = THREE.OneMinusSrcAlphaFactor; //默认值

Blending Equation（混合等式）

THREE.AddEquation THREE.SubtractEquation THREE.ReverseSubtractEquation THREE.MinEquation THREE.MaxEquation

Source Factor（源参数）

THREE.ZeroFactor THREE.OneFactor THREE.SrcColorFactor THREE.OneMinusSrcColorFactor THREE.SrcAlphaFactor THREE.OneMinusSrcAlphaFactor THREE.DstAlphaFactor THREE.OneMinusDstAlphaFactor THREE.DstColorFactor THREE.OneMinusDstColorFactor THREE.SrcAlphaSaturateFactor

Destination Factors（目标参数）

除了 THREE.SrcAlphaSaturateFactor，所有的 Source Factor（源参数）都可作为 Destination Factor（目标参数）。
