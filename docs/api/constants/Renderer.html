WebGLRenderer 常量

Cull Face Modes（面剔除模式）

THREE.CullFaceNone THREE.CullFaceBack THREE.CullFaceFront THREE.CullFaceFrontBack
[page:constant CullFaceNone] 禁用面剔除。
[page:constant CullFaceBack] 剔除背面（默认值）。
[page:constant CullFaceFront] 剔除正面。
[page:constant CullFaceFrontBack] 剔除双面。

Front Face Direction（正面朝向）

THREE.FrontFaceDirectionCW THREE.FrontFaceDirectionCCW
[page:constant FrontFaceDirectionCW] 设置多边形的 winding order（环绕顺序）为顺时针。
[page:constant FrontFaceDirectionCCW] 设置多边形的 winding order（环绕顺序）为逆时针（默认值）。

Shadow Types（阴影类型）

THREE.BasicShadowMap THREE.PCFShadowMap THREE.PCFSoftShadowMap
定义 WebGLRenderer 的 [page:WebGLRenderer.shadowMap.type shadowMap.type] 属性。

[page:constant BasicShadowMap] 使用未过滤的阴影贴图 —— 渲染速度最快，但质量最低。
[page:constant PCFShadowMap] 使用 Percentage-Closer Filtering (PCF) 算法（默认值）过滤阴影贴图。
[page:constant PCFSoftShadowMap] 使用 Percentage-Closer Soft Shadows (PCSS) 算法过滤阴影贴图。

Tone Mapping（色调映射）

THREE.NoToneMapping THREE.LinearToneMapping THREE.ReinhardToneMapping THREE.Uncharted2ToneMapping THREE.CineonToneMapping
定义 WebGLRenderer 的 [page:WebGLRenderer.toneMapping toneMapping] 属性。 用来近似模拟 high dynamic range (HDR)（高动态范围）在普通显示屏或移动设备屏幕等 low dynamic range medium （低动态范围介质）的外观值。

[page:constant NoToneMapping] 禁用 tone mapping（色调映射）。
[page:constant LinearToneMapping] 为默认值。

参考示例 [example:webgl_tonemapping WebGL / tonemapping]。
