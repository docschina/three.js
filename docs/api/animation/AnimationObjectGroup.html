[name]

共享同一动画状态的一组对象

指南的 “下一步” 章节的 “动画系统” 一文中，可查看 three.js 动画系统的各部分的概览。

用法：

添加的对象会作为构造器或 [page:AnimationMixer AnimationMixer] 的 [page:AnimationMixer.clipAction clipAction] 方法的 'root'

这个类的对象是作为混合器的一个整体， 因此各个对象的缓存控制须对整组对象执行。

限制

动画属性必须兼容组中所有对象。

单个属性即可以通过目标组设置，也可以直接设置，但不能同时设置。

构造函数

[name]( [param:object obj1], [param:object obj2], [param:object obj3], ... )

[page:object obj] —— 共享相同动画状态的网格物体。

属性

[property:object stats]

该对象包含此 *AnimationObjectGroup* 的一些信息 （总数，正在使用的数量，每个对象的绑定数）

[property:String uuid]

此 *AnimationObjectGroup* 的 [link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID]。自动分配，不可编辑。

方法

[method:null add]( [param:object obj1], [param:object obj2], [param:object obj3], ... )

向 *AnimationObjectGroup* 添加任意多个对象。

[method:null remove]( [param:object obj1], [param:object obj2], [param:object obj3], ... )

从 *AnimationObjectGroup* 删除任意多个对象。

[method:null uncache]( [param:object obj1], [param:object obj2], [param:object obj3], ... )

将传入的 *AnimationObjectGroup* 的对象所占内存释放掉。
