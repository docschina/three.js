[name]

创建6个相机来渲染到一个 [page:WebGLRenderTargetCube] 对象上。

示例

[example:webgl_materials_cubemap_dynamic materials / cubemap / dynamic ]
[example:webgl_materials_cubemap_dynamic2 materials / cubemap / dynamic2 ]
[example:webgl_shading_physical shading / physical ]
// 创建 cube camera var cubeCamera = new THREE.CubeCamera( 1, 100000, 128 ); scene.add( cubeCamera ); // 创建 car var chromeMaterial = new THREE.MeshLambertMaterial( { color: 0xffffff, envMap: cubeCamera.renderTarget } ); var car = new Mesh( carGeometry, chromeMaterial ); scene.add( car ); // 更新渲染目标 cube car.setVisible( false ); cubeCamera.position.copy( car.position ); cubeCamera.update( renderer, scene ); // 渲染场景 car.setVisible( true ); renderer.render( scene, camera );

构造函数

[name]( [param:Number near], [param:Number far], [param:Number cubeResolution] )

near -- 最近裁剪面的距离（基于相机所在位置，从这一点开始渲染场景）。
far -- 最远裁剪面的距离（基于相机所在位置，一直渲染到场景中这一点）。
cubeResolution -- 设置立方体的宽度。
构造出 1 个包含 6 个 [page:PerspectiveCamera PerspectiveCameras] 的 CubeCamera（全景相机或立方体相机），然后渲染 到 1 个 [page:WebGLRenderTargetCube] 对象上。

属性

通过 [page:Object3D] 基类，可了解相关公共属性。

[property:WebGLRenderTargetCube renderTarget]

生成立方体的纹理。

方法

通过 [page:Object3D] 基类，可了解相关公共方法。

[method:null update]( [param:WebGLRenderer renderer], [param:Scene scene] )

renderer -- 当前的 WebGL 渲染器
scene -- 当前场景
调用此方法来更新渲染 [page:CubeCamera.renderTarget renderTarget] 目标对象。

[method:null clear]( [param:WebGLRenderer renderer], [param:Boolean color], [param:Boolean depth], [param:Boolean stencil] )

调用此方法来清除 [page:CubeCamera.renderTarget renderTarget] 颜色，depth（渲染深度序号，深度值越大则越优先渲染且越靠近观察者）,或 stencil buffer（模版缓冲区）。 颜色 buffer 用来设置清除渲染器的当前颜色。参数默认值为 *true*。
