[name]( )

属性

[property:Float aspect]

默认值是 *1*。

[property:Float eyeSep]

默认值是 *0.064*。

[property:PerspectiveCamera cameraL]

左侧相机（Left camera）。它会被添加到 [page:Layers layer 1] - 要由左侧相机渲染的对象 也必须添加到此 layer。

[property:PerspectiveCamera cameraR]

右侧相机（Right camera）。它会被添加到 [page:Layers layer 2] - 要由右侧相机渲染的对象 也必须添加到此 layer。

方法

[method:null update]( camera )

基于传入的参数相机来更新渲染3D相机（stereo cameras）。
