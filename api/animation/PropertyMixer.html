[name]

缓冲的场景图属性，允许权重累加，内部使用。

构造器

[name]( binding, typeName, valueSize )

-- binding
-- typeName
-- valueSize

属性

[property:Number binding]

[property:Number buffer]

[page:PropertyMixer valueSize] * 4 大小的 buffer。

布局（layout）如下：[ incoming | accu0 | accu1 | orig ]

插值器可以使用 .buffer 作为他们的 .result，然后数据会转到 'incoming'。
'accu0' 和 'accu1' 被用作帧累积结果的帧交织（frame-interleaved）并进行比较以检测变化。
'orig' 存储属性的原始状态。

[property:Number cumulativeWeight]

默认为 *0*。

[property:Number valueSize]

[property:Number referenceCount]

默认为 *0*。

[property:Number useCount]

默认为 *0*。

方法

[method:null accumulate]( accuIndex, weight )

将 [page:PropertyMixer.buffer buffer][accuIndex] 'incoming' 区域中的数据累加到 'accu[i]'。
如果权重为 *0*，则不做任何事情。

[method:null apply]( accuIndex )

当 accus 不同的时候，将 [page:PropertyMixer.buffer buffer] 'accu[i]' 的状态应用于绑定。

[method:null saveOriginalState]( )

记住绑定属性的状态，并将其复制到两个 accus 中。

[method:null restoreOriginalState]( )

将之前通过 'saveOriginalState' 执行的状态应用于绑定。
