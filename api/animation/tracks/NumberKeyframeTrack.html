数值型关键帧值的 track。

构造器

[name]( [param:String name], [param:Array times], [param:Array values] )

[page:String name] —— （必须）KeyframeTrack 的标识符。
[page:Array times] —— （必须）关键帧的 time 数组。
[page:Array values] —— time 数组对应的关键帧的 value 数组。
[page:Constant interpolation] —— 要使用的插值类型。参考 [page:Animation Animation Constants]，默认为 [page:Animation InterpolateLinear]。
