[name]

布尔型关键帧值的 track。

构造器

[name]( [param:String name], [param:Array times], [param:Array values] )

[page:String name] —— （必须）KeyframeTrack 的标识符。
[page:Array times] —— （必须）关键帧的 time 数组。
[page:Array values] —— time 数组对应的关键帧的 value 数组。

属性

参考 [page:KeyframeTrack] 查看继承的属性。

[property:Constant DefaultInterpolation]

默认使用的插值类型，[page:Animation InterpolateDiscrete]。

[property:Array ValueBufferType]

普通的数组（在此处不是 Float32Array 类型，与 [page:KeyframeTrack] 中的 *ValueBufferType* 不同）。

[property:String ValueTypeName]

字符串 'bool'。

方法

参考 [page:KeyframeTrack] 查看继承的方法。

在此处该方法的值为 'undefined'，对离散的属性此方法无意义。
