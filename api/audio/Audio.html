[name]

创建一个 non-positional （无定位的）（全局的）audio（音频）对象。

使用了 [link:https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API Web Audio API]。

示例

[example:webaudio_sandbox webaudio / sandbox ]
[example:webaudio_visualizer webaudio / visualizer ]
// 创建一个 AudioListener 并将其加入到 camera 中 var listener = new THREE.AudioListener(); camera.add( listener ); // 创建全局音频源 var sound = new THREE.Audio( listener ); // 加载声音并设置为 Audio 对象的 buffer（缓冲） var audioLoader = new THREE.AudioLoader(); audioLoader.load( 'sounds/ambient.ogg', function( buffer ) { sound.setBuffer( buffer ); sound.setLoop( true ); sound.setVolume( 0.5 ); sound.play(); });

构造器

[name]( [param:AudioListener listener] )

listener —— （必须）[page:AudioListener AudioListener] 实例。

属性

[property:Boolean autoplay]

是否自动播放。默认为 *false*。

[property:AudioContext context]

构造器中 [page:AudioListener listener] 的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext]。

[property:Array filters]

当前音频是否正在播放。默认为空数组。

[property:GainNode gain]

用 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext/createGain AudioContext.createGain]() 创建的 [link:https://developer.mozilla.org/en-US/docs/Web/API/GainNode GainNode]。

[property:Boolean hasPlaybackControl]

是否可以使用 [page:Audio.play play]()， [page:Audio.pause pause]() 等方法控制播放状态。默认为 *true*。

[property:Number playbackRate]

播放速度。默认为 *1*。

[property:Boolean isPlaying]

当前音频是否正在播放。

[property:Number startTime]

音频播放开始时间。同 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioBufferSourceNode/start AudioBufferSourceNode.start]() 的 *when* 参数一样。默认为 *0*。

[property:Number offset]

音频开始播放时其 buffer 的时间偏移量。 同 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioBufferSourceNode/start AudioBufferSourceNode.start]() 的 *offset* 参数一样。默认为 *0*。

[property:String source]

用 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext/createBufferSource AudioContext.createBufferSource]() 创建的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioBufferSourceNode AudioBufferSourceNode]。

[property:String sourceType]

音频源的类型。默认为字符串 'empty'。

[property:String type]

标识类型的字符串。为 'Audio'。

方法

[method:null connect]()

连接到 [page:Audio.source]。在初始化或设置/移除 filter 时内部使用的方法。

[method:null disconnect]()

断开 [page:Audio.source] 连接。在设置/移除 filter 时内部使用的方法。

[method:Array getFilter]()

返回 [page:Audio.filters filters] 数组的第一个元素。

[method:null getFilters]()

返回 [page:Audio.filters filters] 数组。

[method:Boolean getLoop]()

返回 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioBufferSourceNode/loop source.loop] 的值。 （是否循环播放）.

[method:GainNode getOutput]()

返回 [page:Audio.gain gainNode]。

[method:Number getPlaybackRate]()

返回 [page:Audio.playbackRate playbackRate] 的值。

[method:Number getVolume]( value )

返回当前音量。

[method:null play]()

若 [page:Audio.hasPlaybackControl hasPlaybackControl] 为 true，则播放。

[method:null pause]()

若 [page:Audio.hasPlaybackControl hasPlaybackControl] 为 true，则暂停。

[method:null onEnded]()

播放结束时自动触发。[page:Audio.isPlaying isPlaying] 为 false 时设置。

[method:Audio setBuffer]( audioBuffer )

设置 [page:Audio.source source] 为 audioBuffer，同时 [page:Audio.sourceType sourceType] 为 'buffer'。
若设置了 [page:Audio.autoplay autoplay]，则直接播放。

[method:null setFilter]( filter )

向 [page:Audio.filters filters] 数组中添加 filter。

[method:Audio setFilters]( [param:Array value] )

value —— filters 数组。
设置 [page:Audio.filters filters] 数组为 *value*。

[method:null setLoop]( [param:Boolean value] )

设置 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioBufferSourceNode/loop source.loop] 为 *value* （是否循环播放）。

[method:null setNodeSource]( audioNode )

设置 [page:Audio.source source] 为 audioBuffer，设置 [page:Audio.sourceType sourceType] 为 'audioNode'。
同时设置 [page:Audio.hasPlaybackControl hasPlaybackControl] 为 false。

[method:null setPlaybackRate]( [param:Number value] )

若启用 [page:Audio.hasPlaybackControl hasPlaybackControl]，设置 [page:Audio.playbackRate playbackRate] 为 *value*。

[method:null setVolume]( [param:Number value] )

设置音量。

[method:null stop]()

若启用 [page:Audio.hasPlaybackControl hasPlaybackControl] ，暂停播放， 重置 [page:Audio.startTime startTime] 为 *0* 并设置 [page:Audio.isPlaying isPlaying] 为 false。
