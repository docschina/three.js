[name]

[name] 表示 scene （场景）中所有定位的或无定位的音频的虚拟的 [link:https://developer.mozilla.org/de/docs/Web/API/AudioListener listener]。
通常一个 three.js 应用会创建一个单例 [name]。是 audios entities （音频实体）如 [page:Audio Audio] 和 [page:PositionalAudio PositionalAudio] 构造器的必须参数。
大多数情况下，listener 对象是 camera 的子对象。 所以 camera 的 3D 转换也代表着 listener 的 3D 转换。

示例

[example:webaudio_sandbox webaudio / sandbox ]
[example:webaudio_timing webaudio / timing ]
[example:webaudio_visualizer webaudio / visualizer ]
// 创建一个 AudioListener 并将其加入到 camera 中 var listener = new THREE.AudioListener(); camera.add( listener ); // 创建全局音频源 var sound = new THREE.Audio( listener ); // 加载声音并设置为 Audio 对象的 buffer（缓冲） var audioLoader = new THREE.AudioLoader(); audioLoader.load( 'sounds/ambient.ogg', function( buffer ) { sound.setBuffer( buffer ); sound.setLoop(true); sound.setVolume(0.5); sound.play(); });

属性

[property:AudioContext context]

构造器中 [page:AudioListener listener] 的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext]。

[property:GainNode gain]

使用 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext/createGain AudioContext.createGain]() 创建的 [link:https://developer.mozilla.org/en-US/docs/Web/API/GainNode GainNode]。

[property:AudioNode filter]

默认为 *null*。
