[name]

设置 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext] 所需的方法。

供 [page:AudioListener AudioListener] and [page:AudioLoader AudioLoader] 类内部使用。

使用了 [link:https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API Web Audio API]。

方法

[method:AudioContext getContext]()

若外部作用域定义了变量 *context*，则返回其值， 否则设置为新的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext]。

[method:AudioContext setContext]( [param:AudioConetxt value] )

设置外部作用域的变量 *context* 为 *value*.
