[name]

创建一个 AudioAnalyser 对象，使用 [link:https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode AnalyserNode] 分析音频数据。

使用了 [link:https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API Web Audio API]。

示例

[example:webaudio_sandbox webaudio / sandbox ]
[example:webaudio_visualizer webaudio / visualizer ]
// 创建一个 AudioListener 并将其加入到 camera 中 var listener = new THREE.AudioListener(); camera.add( listener ); // 创建音频源 var sound = new THREE.Audio( listener ); // 加载声音并设置为 Audio 对象的 buffer（缓冲） var audioLoader = new THREE.AudioLoader(); audioLoader.load( 'sounds/ambient.ogg', function( buffer ) { sound.setBuffer( buffer ); sound.setLoop(true); sound.setVolume(0.5); sound.play(); }); // 创建一个 AudioAnalyser，传入音频和 fftSize var analyser = new THREE.AudioAnalyser( sound, 32 ); // 获取音频的平均频率 var data = analyser.getAverageFrequency();

构造器

[name]( audio, [link:https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/fftSize fftSize] )

创建新的 [page:AudioAnalyser AudioAnalyser].

属性

[property:AnalyserNode analyser]

用于分析音频的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode AnalyserNode]。

[property:Integer fftSize]

2 的非零次幂，最大为 2048，表示用于确定 frequency domain （频域）的 FFT (Fast Fourier Transform) 的大小。
详情参考 [link:https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/fftSize this page]。

[property:Uint8Array data]

一个 Uint8Array，其大小由保存分析数据的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/frequencyBinCount analyser.frequencyBinCount] 决定。

方法

[method:Uint8Array getFrequencyData]()

使用 Web Audio 的 [link:https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/getByteFrequencyData getByteFrequencyData] 方法。

[method:Number getAverageFrequency]()

获取音频的平均频率，由 [page:AudioAnalyser.getFrequencyData getFrequencyData] 方法返回。
