尽管 WebGL 兼容性问题越来越微不足道，但一些设备或浏览器依然不能支持 WebGL。 以下方法允许你去检测是否支持 WebGL，并在不能支持 WebGL 时给出提示.

添加 [link:https://github.com/mrdoob/three.js/blob/master/examples/js/Detector.js] 至你的 javascript 并且在你试图渲染之前运行以下代码即可.

if (Detector.webgl) { // 启动方法或其他初始化方法 animate(); } else { var warning = Detector.getWebGLErrorMessage(); document.getElementById('container').appendChild(warning); }
