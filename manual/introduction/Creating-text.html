很多时候你可能需要在 three.js 应用程序中使用文本 —— 本篇将介绍一些常用的方法。

1. DOM + CSS

使用 HTML 通常是添加文本的最简单和最快的方式。 这是大多数 three.js 示例中用于添加描述性文本的方法。

你可以添加内容到一个

<div id="info">Description</div>

同时使用 CSS 标记进行绝对定位，并且通过设置 z-index 属性使其位于其他元素之上， 特别是在全屏运行 three.js 时。

#info { position: absolute; top: 10px; width: 100%; text-align: center; z-index: 100; display:block; }

2. 将文本绘制到画布上并用作为一个 [page:Texture]

如果你希望简单地在 three.js 场景中的平面上绘制文本，请使用此方法。

3. 在你最喜欢的 3D 应用程序中创建一个模型并导出到 three.js

如果你喜欢使用 3D 应用程序（创建模型）并将（创建好的）模型导入到 three.js，请使用此方法。

4. 程序性的文本几何图形

如果你喜欢单纯地使用 three.js 工作，或者创建程序性的动态3D文本几何图形，你可以创建一个 mesh(网格)， 它的 geometry(图形) 是 THREE.TextGeometry 的一个实例：

new THREE.TextGeometry( text, parameters );

为了使其生效，需要在 TextGeometry 的 “font” 参数上设置一个 THREE.Font 的实例。 有关如何做到这一点，每个可接受参数的描述，以及与 three.js 发行版本身一起提供的 JSON 字体列表的更多信息， 请参见 [page:TextGeometry] 页面。

示例

[example:webgl_geometry_text WebGL / geometry / text]
[example:canvas_geometry_text canvas / geometry / text]
[example:webgl_shadowmap WebGL / shadowmap]

如果 Typeface 无法使用，或者你想使用的字体不存在， 有一个用于 blender 的 Python 脚本教程，可以将文本导出成 three.js 支持的的 JSON 格式： [link:http://www.jaanga.com/2012/03/blender-to-threejs-create-3d-text-with.html]
